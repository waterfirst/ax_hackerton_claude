# 판단 기록 (Decision Log)

> 모든 진입/보류/청산의 **이유**를 남긴다. "아무것도 안 한 이유"도 반드시 기록.
> 형식: 날짜 → 시장상황 → 신호 → 판단 → 이유 → 확신도.

- **모델 신호**: 시가 예측 ___ / 종가 예측 ___ / 레짐 ___
- **판단**: BUY / SELL / HOLD / 관망
- **대상·비중**: (슬리브 / 종목 / 금액 / 계좌비중)
- **이유**: (왜 이 판단인가 — 신호 정렬 개수, 근거 3가지)
- **확신도**: 강 / 중 / 약
- **리스크**: (손절선 / 일일한도 / 시나리오 붕괴 조건)
- **결과(마감후 기입)**: 체결여부 / 손익 / 잘한점 / 틀린점
